# Simple Batch OS

![Screenshot](screenshot.png)

A menu-driven operating system interface built with Windows Batch scripting.

## Features

- File management (create, edit, delete files)
- System information monitoring
- Process management
- Network tools
- User management
- Built-in text editor and calculator

## How to Use

1. Download `batch_os.bat`
2. Right-click and select "Run as administrator"
3. Navigate menus using number keys
4. Press Enter to confirm selections
