Simple Batch OS

About This Project

A menu-driven operating system interface built entirely with Windows Batch scripting. This provides a simple UI for common system tasks.

Features

  • File management (create, edit, delete files)
  • System information monitoring
  • Process management
  • Network tools
  • User management
  • Text editor and calculator

Screenshot

How to Use

1. Download the batch file below 2. Right-click and select "Run as administrator" 3. Navigate menus using the number keys 4. Press Enter to confirm selections
